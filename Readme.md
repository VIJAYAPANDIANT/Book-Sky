# 📚 Book Sky - Personal Library Manager

Welcome to **Book Sky**, a sleek and intuitive web application designed to help you organize and manage your personal book collection with ease. Built with modern web technologies, Book Sky offers a premium user experience with a focus on simplicity and functionality.

---

## ✨ Key Features

- **Add New Books**: Easily catalog your books by providing the title, author, and a short description.
- **Dynamic Display**: View your collection in a beautiful, responsive grid layout.
- **Delete Functionality**: Remove books from your list with a single click.
- **Data Persistence**: Your library is stored locally in your browser, so your data remains even after refreshing the page.
- **Responsive Design**: Optimized for various screen sizes, from desktops to mobile devices.
- **Interactive UI**: Enjoy smooth animations, a glassmorphic popup, and a clean "Poppins" typography.

---

## 🛠️ Technology Stack

- **HTML5**: Structured with semantic elements for better accessibility and SEO.
- **CSS3 (Vanilla)**:
  - Custom CSS variables for a consistent theme.
  - Flexbox and CSS Grid for a modern, responsive layout.
  - Interactive hover effects and smooth transitions.
  - Backdrop filters for a premium glassmorphic feel.
- **JavaScript (ES6+)**:
  - DOM manipulation for real-time updates.
  - `localStorage` for client-side data persistence.
  - Event listeners for interactive popups and form handling.

---

## 📂 Project Structure

- `index.html`: The main entry point, containing the structure of the dashboard and the book entry popup.
- `style.css`: The stylesheet that defines the overall aesthetic, including the color palette and responsive behavior.
- `script.js`: The "brain" of the application, handling book list rendering, data storage, and user interactions.

---

## 🚀 Getting Started

To run this project locally, simply follow these steps:

1. **Clone or Download**: Save the project files to your local machine.
2. **Open the App**: Locate the `index.html` file in the project folder.
3. **Launch**: Double-click `index.html` to open it in your favorite web browser.

No servers or complex installations are required!

---

## 📖 Usage Guide

1. **Adding a Book**:
   - Click the vibrant **+** button at the bottom right of the screen.
   - Fill in the **Book Title**, **Author**, and a **Short Description** in the popup.
   - Click **ADD** to save your book to the library.
2. **Deleting a Book**:
   - Locate the book card you wish to remove.
   - Click the red **Delete** button on the card to instantly remove it from your collection.

---

## 💾 Data Persistence

This application uses the browser's `localStorage` API. This means all your book data is stored directly on your device. Even if you close the tab or restart your browser, your library will be waiting for you when you return.
